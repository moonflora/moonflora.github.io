How it works

Moonflora are created by chaotic dynamical systems. The mathematics behind them is quite complex, but the idea is relatively simple.

Take a blank digital canvas and pick a pixel. Now I'm going to roll a dice. The outcome of the dice roll will dictate which pixel I visit next. Now repeat this process many times, and keep track of how often I've visited each pixel. I'm now going to colour each pixel according to how often it was visited. This simple process is how I created every piece in this collection.

What I haven't told you is how we decide to move from pixel to pixel. There is some underlying function, unique to each Moonflora, that specifies this movement. That's the secret sauce! Most functions that you might pick will not give you interesting pictures. I've worked hard to create a special set of functions which result in these floral pieces. I can't tell you what it is though, that would give the game away!

In practical terms, I use python to create all these images. It does the heavy lifting for me - the process of traversing between pixels is done several billion times! I write most of my code from scratch, so rest assured that each piece contains a significant proportion of time and energy from the process of curating the code to be precise, quick and accurate.

Grafting

Grafting is the process of combining your Moonflora to create new child Moonflora, which inherit characteristics from their parents.

As people graft their Moonflora to create new generations, the entire collection will grow and evolve at the hands of the Moonflora community.

The Moonflora collection begins with 1024 Generation 1 Moonflora. Each new generation will have half as many Moonflora, so as the generations get higher, the Moonflora become more rare and more complex

How does Grafting work?

To graft your Moonflora you must purchase a grafting ticket. The grafting ticket allows you to graft a new Moonflora, which will then be airdropped to your wallet. You can only graft Moonflora that you own, and there will be different tickets for each generation (e.g. there will be gen 2 tickets, gen 3 tickets, and so on).

There are a few rules around grafting - these rules are designed so that the collection continues to grow in variety, rather than decline. These rules all relate to the lineage of your Moonflora. The lineage is just the generation 1 Moonflora which appear in a Moonflora's family tree. The lineage also determines which generation your Moonflora lives in: a generation x Moonflora has a lineage of x Moonflora. The golden rule is that grafting must increase the size of the lineage of the Moonflora.
